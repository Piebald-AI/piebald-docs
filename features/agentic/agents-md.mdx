Piebald supports `AGENTS.md` (https://agents.md/).  It will automatically read it from the current directory and attach it to the current chat:

<img class="shadow-md rounded-xl" width="400" alt="Screenshot of a message with AGENTS.md attached" src="/imgs/v0-1-10/agents_md.png" />

## Alternative file names

While `AGENTS.md` is preferred, Piebald will also read from the following other common file names:

- `CLAUDE.md`
- `GEMINI.md`
- `CRUSH.md`
- `QWEN.md`
- `IFLOW.md`
- `WARP.md`
- `copilot-instructions.md`
- `.junie/guidelines.md`
- `.kiro/guidelines.md`
- [+ Add](https://github.com/Piebald-AI/piebald/issues/new?title=Add+new+AGENT.md+alternative+file+name%3a%20)

If you'd like Piebald to support another file name, please [open an issue](https://github.com/Piebald-AI/piebald/issues/new?title=Add+new+AGENT.md+alternative+file+name%3a%20)!
