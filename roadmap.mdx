Here's our immediate public roadmap. We have ideas for many more features besides these that we're planning to explore, but these will help us reach feature parity with existing tools like Claude Code and Gemini CLI.

If there's something not on this list that you want to see added, [let us know](https://github.com/Piebald-AI/piebald/issues/new) and we'll prioritize it!

Core agentic features:

- Plan mode
- Web searching/fetching
- @-mentioning files
- MCP
- Conversation compacting
- Git worktree support
- Sub-agents
- Hooks
- Custom tools
- Edit/command sandboxing

Advanced features:

- Importing chats from Claude Code, Gemini CLI, Codex, Cursor, etc.
- Archiving chats
- Markdown element customizations
- Notifications when sessions are waiting
- Replay streamed responses

Power-user features:

- HTTP & MCP traffic viewer
- Rewriting responses
- Multi-version MCP support
